## Memory Game:
A simple and responsive "Memory Game" built using HTML, CSS, and JavaScript (DOM manipulation).

## Objective:
Flip the cards to find matching pairs. The game ends when all pairs are matched!

## Features:
1. Interactive card flip animation.
2. Randomized (shuffled) cards each time.
3. Match detection logic.
4. Restart button to reset the game.
5. Responsive layout for mobile and desktop.

## Setup:
1. Clone or download this repository.
2. Open "index.html" in any web browser.

## How to Play:
1. Click on any card to flip it.
2. Try to remember its position and symbol.
3. Match two cards with the same symbol.
4. All cards stay flipped when matched.
5. Use the “Restart Game” button to play again.

## Technologies Used:
HTML5 — Structure  
CSS3 — Styling and animations  
JavaScript — Game logic
